# Component Overriding

material-table has many components inside itself. You can change some of them according to your needs.

## Usage

In this usage developer changes backgroundColor of Toolbar component.

```jsx
import MaterialTable, { MTableToolbar } from 'material-table';

<MaterialTable
    // other props
    components={{
        Toolbar: props => (
            <div style={{ backgroundColor: '#e8eaf5' }}>
                <MTableToolbar {...props} />
            </div>
        )
    }}
/>
```
