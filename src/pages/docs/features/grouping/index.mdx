# Grouping

material-table has grouping feature that lets users to group data by one or more columns.

## Usage

To make grouping available you must open grouping in options.

```jsx
import MaterialTable from 'material-table';

<MaterialTable
    // other props
    options={{
      grouping: true
    }}
/>
```

<br />

## Hints 

* You can make a column default grouped by setting `column.defaultGroupOrder` with order column and setting `column.defaultGroupSort` with `asc` or `desc`
* You can close grouping for any column by setting `column.grouping` as `false`
* You can override grouping bar component with overriding `components.Groupbar`
* You can override grouped row component with overriding `components.GroupRow`
