# Custom Column Rendering

material-table lets you to override a render of any column. For example, you can render an image instead of image url.

## Usage

In this usage developer changes backgroundColor of Toolbar component.

```jsx
import MaterialTable, { MTableToolbar } from 'material-table';

<MaterialTable
    // other props
    columns={[
      {
        field: 'url',
        title: 'Avatar',
        render: rowData => <img src={rowData.url} style={{width: 50, borderRadius: '50%'}}/>
      }
    ]}
/>
```
