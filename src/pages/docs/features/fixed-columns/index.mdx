# Fixed Columns

material-table has fixed columns feature both left and right. 

## Usage

To use fixed columns feature you should add fixedColumns object to options

```jsx
import MaterialTable from 'material-table';

<MaterialTable
    // other props
    options={{
      fixedColumns: {
        left: 2, 
        right: 0
      }
    }}
/>
```

<br />

## Hints 

* TableLayout would be `fixed` if you add any fixed columns

> You can not use fixed columns feature with detail panels, treedata or grouping for now.
