# Filtering

material-table has inline filtering feature that lets users to filter every column separately.

## Usage

To make filtering available you must open filtering in options.

```jsx
import MaterialTable from 'material-table';

<MaterialTable
    // other props
    options={{
      filtering: true
    }}
/>
```

<br />

## Hints 

* You can add a default filter value to a column by setting `column.defaultFilter`
* You can close filtering for any column by setting `column.filtering` as `false`
* You can override filtering component with overriding `components.FilterRow`
* You can change filter icon with overriding `icons.Filter`
